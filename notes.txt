RxJS
-----
-> RxJS is a library for building asynchronous applications with observable sequences.

RxJS Building Blocks
---------------------
Observables
Observers
Subscriptions
Operators
Subjects
Schedulers

Observable
-----------
-> A representation of any set of values over any amount of time

Observers
---------
-> Objects with simple interface
    -next()
    -error()
    -complete()
